# node-lambda


Simple example of a webhook using  [Amazon Lambda](http://aws.amazon.com/lambda/).


```
$ sls deploy
```

## Installation

```
$ aws configure
```
```
$ npm install
```
```
$ sls deploy
```


## Usage

You can use this website.

